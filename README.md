# LA Live Cams

Современное приложение для просмотра вебкамер Лос-Анджелеса с интерактивной картой.

## Стек технологий
- React + Vite
- Tailwind CSS (Стилизация)
- React Leaflet (Карты)
- React Player (Видео)

## Установка и запуск

```bash
npm install && npm run dev
```

Открыть в браузере ссылку, предложенную Vite (обычно http://localhost:5173).
